Django Template Engine

Django's template engine allows you to generate dynamic HTML pages using Python data passed through the context.

Blocks

Blocks are placeholders defined in a base template and overridden in child templates. Example:

{% block content %}...{% endblock %}

For Loops

Used to iterate over lists or querysets:

{% for item in items %}
  <p>{{ item }}</p>
{% endfor %}

If Statements

Used for conditional rendering:

{% if user.is_authenticated %}
  <p>Welcome, {{ user.username }}!</p>
{% else %}
  <p>Please log in.</p>
{% endif %}

Context Variables

Data is passed from the view to the template via a context dictionary:

return render(request, "template.html", {"name": "Alice"})

Then in the template, you can access it with {{ name }}.
